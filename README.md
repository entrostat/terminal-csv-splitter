# Usage
<!-- usage -->
```sh-session
$ npm install -g terminal-csv-splitter
$ terminal-csv-splitter COMMAND

$ terminal-csv-splitter (-v|--version|version)
terminal-csv-splitter/0.0.0 linux-x64 node-v14.15.0
$ terminal-csv-splitter --help [COMMAND]
USAGE
  $ terminal-csv-splitter COMMAND
...
```
